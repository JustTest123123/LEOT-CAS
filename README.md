<h3 id="XlyaP">**5. 注意事项**</h3>
+ **存储配置**：需要进一步讨论 `.std` 文件格式、文件名规则（是否包含角度信息等）。
+ **IDL弹窗问题**：需要确认IDL调用时的弹窗是否会影响自动化流程。
+ **光谱饱和判断逻辑**：需要明确判断条件和触发机制。
